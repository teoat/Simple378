Backend Deployment:
  1. docker build -f backend/Dockerfile -t simple378-backend:latest .
  2. docker push simple378-backend:latest
  3. docker run -e TENANT_PLAN=enterprise simple378-backend:latest

  3. docker push simple378-frontend:latest
  4. Service worker will auto-register on first visit

  3. Check for PWA install banner at top of page
